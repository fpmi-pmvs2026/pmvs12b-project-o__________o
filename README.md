# WeatherApp

## Description
WeatherApp – это современное Android-приложение для отслеживания погоды в реальном времени. Приложение использует внешний API (OpenWeatherMap) для получения актуальных данных, сохраняет избраные города в локальной базе данных Room и отправляет push-уведомления при резких изменениях погоды. Интерфейс построен на Jetpack Compose, что обеспечивает плавную анимацию и адаптивность под разные экраны. Приложение поддерживает многопоточную загрузку данных, оффлайн-режим и настраиваемые уведомления через WorkManager.

## Installation
1. Клонируйте репозиторий
2. Перейдите в папку с кодом
3. Откройте проект в Android Studio (Flamingo или новее).
4. Синхронизируйте Gradle: `File -> Sync Project with Gradle Files`
5. Запустите приложение на эмуляторе или устройстве (API 24).

## Usage
- При первом запуске разрешите доступ к геолокации (опционально) или введите город вручную.
- Добавьте города в избранное свайпом или кнопкой `+`.
- Настройте уведомления в разделе `Settings` (порог температуры, тип осадков).
- Данные автоматически обновляются каждые 30 минут или при открытии приложения.
- Оффлайн-режим работает автоматически: последние сохранённые данные отображаются без интернета.

## Contributing
Проект разработан командой из 3 (1) человек:
- **Тимлид / Backend & API**: Интеграция Retrofit, обработка JSON, настройка GitHub Actions, CI/CD.
- **UI/UX & Compose**: Разработка экранов на Jetpack Compose, анимации, адаптивная вёрстка.
- **Data & Notifications**: Room Database, WorkManager для фоновых задач, push-уведомления, unit/UI тесты.

Все задачи распределялись через GitHub Projects. Ветки создавались по шаблону `feature/`, `bugfix/`, `docs/`. Мерж в `main` осуществлялся через Pull Request после прохождения тестов.
